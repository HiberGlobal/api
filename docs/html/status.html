MaintenanceMode

The maintenance modes

NameNumberDescription
NONE 0

System operating normally.
RELEASE 1

Release is in progress, some customer operations may be blocked.
READ_ONLY_PREPARATION 2

Preparing for full read-only mode. This blocks new requests but lets the ongoing ones be handled.
READ_ONLY 3

Read-only mode. No changes can be made.
